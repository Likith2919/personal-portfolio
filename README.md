# Personal Portfolio - Likith Sirigiri

Senior DevOps and Platform Engineering portfolio built with React + Vite.

## Highlights

- Dual language support (English and Russian)
- Recruiter-focused sections: impact metrics, case-study cards, experience timeline
- Hiring intent aligned to Russia opportunities (remote/onsite)
- Downloadable resume from `public/DevOps Engineer.pdf`

## Tech Stack

- React 19
- Vite 7
- Framer Motion
- Lucide Icons

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Update Before Applying

- Add your GitHub URL in `src/App.jsx` (`profile.github`)
- Add your LinkedIn URL in `src/App.jsx` (`profile.linkedin`)
- Replace case-study links in `src/App.jsx` (`idp[].href`) with project URLs
